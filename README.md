# Kata Roman Numbers

Write a method `convert(number): string` that takes a number and converts it to the according `string` representation.

```
   1 ➔ I
   2 ➔ II
   3 ➔ III
   4 ➔ IV
   5 ➔ V
   9 ➔ IX
  21 ➔ XXI
  50 ➔ L
 100 ➔ C
 500 ➔ D
1000 ➔ M
```

## Steps to start the kata
- `npm install`
- `npm run test` or `npm run test-watch`
